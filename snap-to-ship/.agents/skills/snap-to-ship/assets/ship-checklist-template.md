# Ship Checklist (Template)

## Functionality
- [ ] Can navigate primary routes
- [ ] Primary CTA works (even if mocked)
- [ ] Form validation present (if forms exist)
- [ ] Modal open/close works (if used)

## States
- [ ] Loading
- [ ] Empty
- [ ] Error

## UX
- [ ] Mobile breakpoint tested
- [ ] Keyboard navigation reasonable
- [ ] Buttons have clear labels
- [ ] Inputs have labels and helpful placeholders

## Engineering
- [ ] `npm run lint`
- [ ] `npm run build`
- [ ] No secrets committed
- [ ] `.env.example` present if env vars used
