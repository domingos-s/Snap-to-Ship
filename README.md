Turn screenshots into working apps using Codex.

## Install
Copy this skill into any repo:

.agents/skills/snap-to-ship/

Or clone this repo and copy the .agents folder.

## Use inside Codex
Open Codex in your repo and type:

/skills

Then run:

$snap-to-ship

Upload a UI screenshot and choose:
- prototype mode (fast UI)
- mvp mode (auth + DB + CRUD)

Codex will generate a full Next.js + Tailwind app.
